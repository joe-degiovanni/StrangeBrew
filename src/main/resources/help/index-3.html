Getting started

This section contains the information you need to install StrangeBrew, start the software, and get technical support.

Installing StrangeBrew

Before you install StrangeBrew, ensure you have enough free disk space. StrangeBrew requires about 2.5 MB of free space.

To install StrangeBrew:

  1. Unzip the StrangeBrew install file (sb_java.zip) with an unzip utility such as Winzip.

Starting the software

You can start StrangeBrew one of two ways:

  1. From the command-line, type java -jar strangebrew_fat.jar.

  2. Double-click the StrangeBrew .jar file (strangebrew_fat.jar). If .jar files are associated with the Java runtime, this will launch the application.

  3. You may also need to install the One Jar library from SourceForge to run the strangebrew_fat.jar file. This seems to be the case with Mac OS-X in particular.

Check the readme file for more information about starting the application, including additional help for setting up Java on Linux, and how you can set the Java Look and Feel.

Setting your preferences

If this is the first time you have run StrangeBrew you should set your preferences before you create new recipes. You can set preferences for:

  • new recipe defaults for units, recipe size, and mash specifications

  • cost defaults

  • calculations

  • screen appearance

For more information, see See Setting StrangeBrew Preferences.

Getting technical support

All users can receive technical support for StrangeBrew. The following technical support resources are available:

  • StrangeBrew help and user guide - you're reading it! Check this guide to see if it answers your question.

  • StrangeBrew 2.0 user forum - on the internet at http://www.sourceforge.net/projects/strangebrew

  • Email support - send your questions to support@strangebrew.ca
