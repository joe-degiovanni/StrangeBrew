Editing StrangeBrew Databases

StrangeBrew relies on these databases for all its ingredients and calculations:

  • The Malts Database

    : a list of malts, adjuncts, sugars, and fruits you can add to a recipe.

  • The Hops Database

    : a list of hops

  • The Yeast Database

    : a list of brewing yeasts

  • The Miscellaneous Ingredient Database

    : a list of miscellaneous ingredients, such as finings, spices, weird Papazian-esque stuff, and water treatments.

  • The Styles Database

    : a list of recognized BJCP styles.

You can edit any of the databases in the Data directory using a standard text editor, or a spreadsheet program that understands CSV (comma-separated values) format.
